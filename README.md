# Onda

> Premium motion graphics components for [Remotion](https://remotion.dev) — installed as source, owned by you.

Onda is an open-source library of motion graphics components built on Remotion. Components are not imported as a black-box dependency — you install them with the `onda` CLI and the source lands in your project, yours to own and edit.

```bash
npx onda add blur-reveal
```

## Why Onda

Onda's edge is a **signature motion identity** — a recognizable way everything moves, applied across a clean set of components. Think Apple, Linear, or Stripe motion: identifiable by feel before any logo appears. Uniqueness lives in *how it moves*, not *what it is*.
